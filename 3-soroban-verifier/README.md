# Soroban ZK Verifier (UltraHonk)

This project contains a Soroban Smart Contract that can verify Zero-Knowledge Proofs generated by Noir using the UltraHonk backend.

## üîÑ Interaction Flow

1.  **Generate Proofs**: Use the `1-noir-circuit` to generate the `proof`, `public_inputs`, and verification key (`vk`).
    *   **Important**: You must generate artifacts using `ultra_honk` scheme and `keccak` oracle hash.
    *   Example: `bb write_vk --scheme ultra_honk --oracle_hash keccak ...`
2.  **Deploy**: Deploy this contract to Stellar (Localnet/Testnet).
    *   Constraint: You must pass the Verification Key (VK) to the `__constructor`.
3.  **Verify**: Invoke `verify_proof` with:
    *   `public_inputs`: The public inputs from your circuit (e.g., price, threshold).
    *   `proof_bytes`: The cryptographic proof.
